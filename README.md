# easy_adb_android
Basic batch file to quick connect adb over Wifi or change server settings
### Options: <br>
 *Check connected devices <br>
 *Kill server<br>
 *Start CAST w console<BR>
 *Start CAST w/o console<br>
 *Set TCP/IP port<br>
 *Connect to ip<br>
 *Exit<br>
STRCPY Modules are derived<br>
